# Geometry Viewer

The Geometry Viewer is a standalone single page web application to visualize and explore VTP files. The only requirement is the single HTML file without any web server. This application can be use to load vtp files and use geometry rendering to render them.
The current vtp supported format is ascii, binary and binary+zlib compression.

Several sample .VTP files can be found here.

You can specify remote VTP links to visualize with Geometry Viewer as extra arguments to the URL such as in the links below:

  • diskout.vtp 471.9 kB
  • diskout-stream-ascii.vtp 1.13 MB
  • diskout-stream-binary.vtp 844.4 kB
  • diskout-stream-binary-zlib.vtp 501.3 kB
  • Earth.vtp 1.17 MB
  • Brain Blood Vessels 57.71 MB
  • Chest CT 63.75 MB

WebXR Examples

Virtual reality, augmented reality, and holographic viewing is supported for WebXR devices using the xrSessionType URL parameter.

Virtual Reality

For head-mounted devices such as Meta Quest 2, HP Reverb G2, or Microsoft HoloLens 2.

  • Earth.vtp (VR headset) 1.17 MB
  • Brain Blood Vessels (VR headset) 57.71 MB
  • Chest CT (VR headset) 63.75 MB

Handheld Augmented Reality

For handheld devices such as Android devices or iPhones (see iPhone requirements).

  • diskout.vtp (handheld AR) 471.9 kB
  • Brain Blood Vessels (handheld AR) 57.71 MB
  • Chest CT (handheld AR) 63.75 MB

Augmented Reality Passthrough Headset

For head-mounted devices (HMDs) with augmented reality passthrough support such as Meta Quest 2.

  • Earth.vtp (AR headset) 1.17 MB
  • Brain Blood Vessels (AR headset) 57.71 MB
  • Chest CT (AR headset) 63.75 MB

Holographic

For Looking Glass Factory holographic displays.

  • Brain Blood Vessels (holographic) 57.71 MB
